# Grocereo Online Website

Static website for grocereo.online.

## Files
- `index.html` - homepage
- `team.html` - team page linked from the Company menu
- `assets/styles.css` - responsive styling
- `assets/script.js` - mobile navigation
- `assets/images/` - optimized team photos

## Deploy to GitHub Pages
1. Upload all files to your GitHub repository.
2. In GitHub, go to Settings > Pages.
3. Select the main branch and root folder.
4. Add the custom domain `grocereo.online` in the Pages settings.
